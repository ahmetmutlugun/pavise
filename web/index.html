Analysis Coverage

Six layers of
security inspection

Every IPA is unpacked and analyzed across binary, manifest, resources, and source-level patterns.

Binary Analysis

Mach-O parsing for PIE, ARC, stack canaries, encryption flags, and symbol-level inspection.

Secret Detection

Regex-based scanning for hardcoded API keys, tokens, passwords, and embedded credentials.

OWASP Mobile Top 10

Full M1-M10 coverage with CWE references for compliance reporting.

Tracker SDKs

Identifies analytics, advertising, and attribution SDKs embedded in the binary.

Transport Security

ATS exceptions audit, insecure domain detection, and network security configuration review.

Supply Chain

Framework software composition analysis with known CVE matching from vulnerability databases.
