Pavise

Fast mobile app static security analysis

What it does

Pavise performs deep static analysis of iOS IPA and Android APK files in under 3 seconds. It inspects binaries, manifests, embedded resources, and source patterns to surface security issues before they ship.

Analysis coverage

+ Binary protections (PIE, ARC, stack canaries)
+ Hardcoded secrets and API keys
+ OWASP Mobile Top 10 mapping
+ Third-party tracker detection
+ Transport security (ATS) audit
+ Framework SCA with CVE lookup
+ Entitlements and permissions
+ Firebase and cloud misconfig
+ URL and email extraction
+ Provisioning profile analysis

Output formats

JSON for automation and CI/CD pipelines. PDF for stakeholder reports. SARIF 2.1.0 for IDE integration. HTML for quick review.
